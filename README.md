# JTools

A collection of simple Java command-line tools for everyday tasks.

These tools are operating system independent and work on macOS, Linux, and Windows (unless specified explicitly for a given tool).

## Tools

- **Duplicates** - Remove duplicate entries from PATH or any other colon/semicolon-separated environment variable
- **Find** - Search for files and directories by name with highlighted results

## Usage

Run any tool directly with Java:

```bash
java src/main/java/michalmm/jtools/Find.java <searchTerm>
```
